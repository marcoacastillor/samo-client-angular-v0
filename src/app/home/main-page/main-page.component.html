Sales System

Sistema de gestión y control de ventas

SalesSystem es un software fácil y sencillo, diseñado para pymes donde puedes llevar día a día y desde cualquier lugar, la administración de su información. El sistema ofrece soporte para los procesos de ventas, compras, inventarios, procesos productivos; incluyendo reportes contables de forma eficiente y segura.

Arquitectura

SalesSystem fue diseñado bajo un enfoque de arquitectura distribuida y orientada a web, para ofrecer el acceso a la información en tiempo real y con una alta disponibilidad.

Se desarrollo bajo tecnologías de punta, para ofrecer una mejor experiencia de usuario, seguridad, accesibilidad y escalabilidad cuando el sistema, así se requiera.

Tecnología

Para la construcción del sistema se eligió para desarrollo de la capa de presentación a AngularJS versión 6 y para la lógica de negocio se escogió a PHP-Laravel versión 5.1, y se exponen recursos REST para dar accesibilidad a múltiples clientes.

Módulos del sistema

Compras

El módulo de compras permite una gestión organizada de todas las compras realizadas por la empresa, permite llevar un control sobre las compras por período de tiempo, cuentas por pagar, estado de cuenta por proveedor, tiempo de recaudo de facturas, entre otras funcionalidades.

Ventas

El módulo de ventas permite una gestión organizada de todas las facturas generadas, permite llevar un control sobre las ventas por período de tiempo, cuentas por cobrar, estado de cuenta por clientes, tiempo de recaudo de facturas, entre otras funcionalidades.

Inventarios

El módulo de inventarios gestiona la información de todos los productos que se puedan comercializar, ya sean; productos comprados a terceros o producidos por la empresa. Igualmente permite llevar un control de inventarios, generación de código de barras propios y reportes de productos.

Recurso Humano

Módulo que permite llevar un registro sobre todas las personas que tienen algún vínculo con la empresa, pueden ser clientes o trabajadores, para los trabajadores, permite llevar un registro sobre contratación, novedades de nómina, pago de salario, entre otras funcionalidades.

Procesos de producción

Este módulo, permite llevar un control sobre los productos que produce la empresa para comercializar, llevando un control sobre los insumos necesarios en cada proceso productivo, dividiendo el proceso en períodos de corte para un mayor control sobre los productos, tanto resultantes como insumos utilizados en cada período.

Liquidación

Módulo que permite llevar un control sobre las liquidaciones realizadas dado un período, para cada trabajador de la empresa, el sistema permite generar pagos por unidad productiva o salario fijo.

Egresos

Módulo que gestiona los gastos realizados por la empresa dia a dia, permite llevar una numéración consecutiva y/o facturas equivalentes.

Empresas

En el módulo de empresas, se lleva un control sobre todas las empresas proveedoras con las que la empresa tiene alguna relación, permitiendo hacer un seguimiento histórico de compras y cuentas por pagar.

Administración y gestión de acceso

Toda la administración de usuarios, roles y permisos para usuarios, se realiza en éste módulo. Se garantiza la seguridad de la información contenida en el sistema, dado que implementa un sistema de acceso por token bajo encriptación SSH con algoritmo de cifrado SHA156. Se configura un tiempo de expiración del token para renovar accesos y validar seguridad.
